Assessment Center Inklusif

Petunjuk Pelaksanaan ACI

  1. Ini hanya diperuntukkan untuk yang kemarin berhalangan mengikuti test assessment Manager maupun Senior Manager di Head Quarter (kantor pusat).
  2. Ada 2 bagian di sini, sisi kiri untuk Manager dan sisi kanan untuk Senior Manager. Silakan dipilih sesuai posisi jabatan Anda.
  3. Ikuti flow nya: klik Test TKD → lanjut ke Test TLMS → setelah itu Essay.
  4. Jika ada pertanyaan atau kesulitan, silakan hubungi tim Assessment.
